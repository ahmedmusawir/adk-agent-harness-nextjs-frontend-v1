## Lessons Learned

  over it — the mount effect's blind `setAgentSessions(fetched)` would have
  silently nullified the persist wrapper.
- When jsdom can't simulate an environment condition (no `window`), test the
  code path the condition produces (storage getter throws), not the condition.
